# Game

## Página del juego
La página del juego está en `index.html` y se sirve desde el portal local.

- URL del portal: [http://127.0.0.1:8080](http://127.0.0.1:8080)

## Ejecutar juego directamente
```bash
pip install pygame
python juego.py
```

## Portal web local para acceder al juego
```bash
python portal.py
```
Luego abre: `http://127.0.0.1:8080`

Desde la página podrás pulsar **Lanzar juego** para abrir `juego.py`.
